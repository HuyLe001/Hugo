## Week 2 Achievements

• Learned EBS (Elastic Block Store) volumes including creation, attachment, and snapshot management.  
• Understood VPC fundamentals: Subnets, CIDR blocks, Route Tables, Internet Gateway, and NAT Gateway.  
• Successfully created basic VPC setup and practiced VPC networking configuration.  
• Learned difference between Security Groups and NACLs, configured inbound/outbound rules.  
• Got introduced to RDS (Relational Database Service) and various database engines (MySQL, PostgreSQL).  
• Understood benefits of managed databases and basic DynamoDB (NoSQL) concepts.  
• Learned Load Balancers (ELB and ALB) and basic Auto Scaling concepts for high availability.  
• Improved AWS documentation reading skills and console navigation.
