## Kết quả đạt được tuần 1

• Kết nối với các thành viên First Cloud Journey và hoàn thành hướng dẫn bootcamp.  
• Học các khái niệm nền tảng AWS cloud computing, thuật ngữ cơ bản (Regions, Services, IaaS/PaaS/SaaS), và Hạ tầng Toàn cầu của AWS.  
• Tạo thành công tài khoản AWS Free Tier và cấu hình Virtual MFA Device để bảo mật.  
• Hiểu các khái niệm IAM cơ bản (Users, Groups, Policies) và tạo admin group và IAM user đầu tiên.  
• Làm quen với việc điều hướng AWS Management Console và vị trí các dịch vụ.  
• Cài đặt và cấu hình AWS CLI với thông tin xác thực (Access Key, Secret Key, Region).  
• Thực hành các lệnh AWS CLI cơ bản: `aws configure`, `aws s3 ls`, `aws iam list-users`.  
• Học các khái niệm cơ bản về EC2 bao gồm instance types (t2.micro), AMI, và Security Groups.  
• Khởi tạo thành công EC2 instance đầu tiên và thực hành kết nối SSH.  
• Hiểu các khái niệm cơ bản về S3 (Buckets và Objects), tạo S3 bucket, và thực hành upload/download file.
