## Kết quả đạt được tuần 6

• Học các khái niệm cơ bản về API Gateway và REST API (resources, methods, endpoints).  
• Tạo CRUD REST API hoàn chỉnh với API Gateway tích hợp Lambda functions.  
• Cấu hình CORS cho browser access và API deployment stages (dev, prod).  
• Xây dựng serverless stack hoàn chỉnh: API Gateway → Lambda → DynamoDB cho task management API.  
• Test APIs sử dụng Postman và học về HTTP status codes và error handling.  
• Hoàn thành hiểu biết về full serverless architecture từ frontend đến backend integration.  
• Hiểu các tính năng API Gateway bao gồm request/response transformations và API deployment stages.  
• Học cơ bản về API authentication (API Keys, IAM) và tầm quan trọng của error handling đúng cách.
