**Week 1:** [Introduction & Environment Setup](1.1-week1/)  
Understanding Cloud computing models, Regions, and Availability Zones. Creating and configuring AWS account. Introduction to Free Tier, cost management, and Budget alerts. Getting familiar with AWS Management Console and AWS CLI.

**Week 2:** [Compute & Networking Basics](1.2-week2/)  
Services: EC2, Elastic IP, Auto Scaling, VPC basics. Network components: Subnet, Route Table, Security Group, NACL. Hands-on: Deploying a web server on EC2 accessible from the internet.

**Week 3:** [Storage & Database](1.3-week3/)  
Services: S3, EBS, EFS, RDS. Differentiating block / object / file storage. Hands-on: Creating S3 bucket with versioning & lifecycle, connecting application to RDS.

**Week 4:** [IAM & Basic Security](1.4-week4/)  
Services: IAM Users, Groups, Roles, Policies, MFA. AWS account security best practices. Hands-on: Creating users following least privilege principle, testing permissions.

**Week 5:** [Monitoring & Basic Automation](1.5-week5/)  
Services: CloudWatch, CloudTrail, SNS, EventBridge. Introduction to Infrastructure as Code (IaC) – CloudFormation basics. Hands-on: Creating EC2 + S3 Stack using CloudFormation.

**Week 6:** [Foundation Review + Mini Project](1.6-week6/)  
Review of all services: EC2, S3, RDS, IAM, CloudWatch. Deploying mini project: Static website or small web app using S3 + EC2 + RDS. Result: Solid foundation project and knowledge before entering specialized tracks.

**Week 7:** [DevOps & Automation](1.7-week7/)  
CI/CD with CodeCommit, CodeBuild, CodeDeploy, CodePipeline. Advanced IaC: CloudFormation, AWS CDK, Terraform (optional). Containers: ECS, ECR, EKS. Hands-on: Automating web app build & deployment using CodePipeline.

**Week 8:** [Container & Orchestration](1.8-week8/)  
Deep dive into container services: ECS, EKS, Fargate. Container networking and service discovery. Hands-on: Deploying containerized applications with ECS and EKS.

**Week 9:** [Security & Compliance](1.9-week9/)  
Zero Trust architecture. Key management with AWS KMS, Secrets Manager. S3 data security, logging, and encryption. Hands-on: Setting up security workflow with IAM + KMS + CloudTrail.

**Week 10:** [Serverless & Modern Applications](1.10-week10/)  
Serverless services: Lambda, API Gateway, DynamoDB, AppSync. Building event-driven architectures. Hands-on: Mini project - Serverless API or real-time application.

**Week 11:** [Final Project Development](1.11-week11/)  
Designing complete architecture (front-end → backend → database). Applying CI/CD, security, monitoring, and cost optimization. Result: Completed real-world project ready for CV portfolio.

**Week 12:** [Project Showcase & Career Prep](1.12-week12/)  
Writing blog post / tech talk to share project experience. Reviewing and practicing for AWS Certified Cloud Practitioner exam. Preparing CV, LinkedIn profile, and cloud interview skills. Participating in FCJ demo day / showcase to present the final product.
