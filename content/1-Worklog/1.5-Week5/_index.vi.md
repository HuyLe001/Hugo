## Kết quả đạt được tuần 5

• Học về S3 versioning, lifecycle policies và storage classes (Standard, Intelligent-Tiering, Glacier).  
• Giới thiệu về DynamoDB - tạo NoSQL table đầu tiên và hiểu thiết kế partition/sort key.  
• Thực hành Lambda functions với các triggers khác nhau (S3 events, DynamoDB streams, API calls).  
• Xây dựng ứng dụng serverless: S3 upload → Lambda processing → DynamoDB metadata storage.  
• Hiểu sự khác biệt giữa Query vs Scan operations trong DynamoDB để truy xuất dữ liệu.  
• Giới thiệu về Infrastructure as Code (IaC) với CloudFormation cơ bản và cấu trúc template.  
• Hiểu sâu hơn về các mẫu kiến trúc serverless và event-driven workflows.  
• Cải thiện kỹ năng cấu hình IAM role và học cách debug Lambda functions sử dụng CloudWatch Logs.  
• Nhận ra serverless có thể tiết kiệm chi phí cho các ứng dụng nhỏ.
