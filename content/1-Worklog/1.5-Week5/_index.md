## Week 5 Achievements

• Learned S3 versioning, lifecycle policies, and storage classes (Standard, Intelligent-Tiering, Glacier).  
• Introduction to DynamoDB - created first NoSQL table and understood partition/sort key design.  
• Practiced Lambda functions with different triggers (S3 events, DynamoDB streams, API calls).  
• Built serverless application: S3 upload → Lambda processing → DynamoDB metadata storage.  
• Understood difference between Query vs Scan operations in DynamoDB for data retrieval.  
• Introduction to Infrastructure as Code (IaC) with CloudFormation basics and template structure.  
• Gained deeper understanding of serverless architecture patterns and event-driven workflows.  
• Improved IAM role configuration skills and learned to debug Lambda functions using CloudWatch Logs.  
• Realized serverless can be cost-effective for small applications.
