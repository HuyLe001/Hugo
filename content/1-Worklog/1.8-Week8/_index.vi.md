## Kết quả đạt được tuần 8

• Học CI/CD fundamentals bao gồm Continuous Integration và Continuous Deployment principles.  
• Thực hành Git version control cơ bản với AWS CodeCommit repositories.  
• Tạo buildspec.yml configuration files cho CodeBuild để compile và test applications.  
• Cấu hình CodeDeploy cho automated application deployments đến các targets khác nhau.  
• Build complete CI/CD pipeline với CodePipeline: CodeCommit → CodeBuild → CodeDeploy automation.  
• Giảm thời gian manual deployment từ 30 phút xuống 5 phút thông qua pipeline automation.  
• Hiểu lợi ích của automation bao gồm deployments nhanh hơn, ít lỗi hơn và processes nhất quán.  
• Đạt được practical DevOps skills và hiểu tầm quan trọng của automated testing và deployment strategies.
